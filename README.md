#  Student Result System (Java)

##  Overview
A simple Java program to process student marks using **control flow**. Demonstrates `if-else`, `switch`, loops, `break`, `continue`, and methods for clean code.

##  Features
- Input marks for multiple students
- Validation (0–100 range)
- Grade assignment via `switch`
- Loop until user exits (`-1`)
- Skip invalid entries with `continue`
- Refactored methods for readability

##  Grading
| Percentage | Grade |
|------------|-------|
| 90–100     | A     |
| 80–89      | B     |
| 70–79      | C     |
| 60–69      | D     |
| <60        | F     |
